# Welcome to Tourmaline
## What is Tourmaline?

Tourmaline is a simple, elegant programming language designed to be easy to learn while remaining powerful enough for real projects. With syntax inspired by Lua and Ruby, Tourmaline offers a clean and intuitive way to write code.

## Quick Example

```python
# A simple Tourmaline program
let name = input("What's your name? ")
print("Hello, " + name + "!")

function greet(person, times)
    let i = 1
    while i <= times
        print("Welcome, " + person + "!")
        i += 1
    end
end

greet(name, 3)
```

## Why Choose Tourmaline?

- **Beginner-Friendly**: Simple syntax that doesn't sacrifice functionality
- **Expressive**: Write what you mean in a natural, readable way
- **Practical**: Built-in functions for math, lists, strings, and more
- **Open Source**: Free to use under the GNU GPL v3.0 license
- **Cross-Platform**: Runs anywhere Python runs
